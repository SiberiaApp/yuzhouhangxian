# 宇宙航线：REAREye独家
[![](https://gh-proxy.org/https://raw.githubusercontent.com/SiberiaApp/yuzhouhangxian/refs/heads/main/haibao.png)](https://gh-proxy.org/https://raw.githubusercontent.com/SiberiaApp/yuzhouhangxian/refs/heads/main/haibao.png)

从此刻的坐标出发，向未知的星域航行。以星图与轨道为灵感，构建「宇宙航线」。通过留白与轨道元素的呼应，在保留信息显示空间的同时，营造简洁而富有秩序感的画面。

与小米7月背屏创意设计大赛入围作品——宇宙航线作者hrsthrt74独家合作，实现了AOD界面下显示深色版本，并首次以MAML的形式实现了背屏充电动画，仅在REAREye独家上线！
